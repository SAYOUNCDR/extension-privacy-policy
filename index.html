Privacy Policy for ChatGPT Favorites

Effective Date: December 2025

ChatGPT Favorites is a Chrome extension that allows users to bookmark or favorite important ChatGPT conversations directly in the sidebar.

Download from Chrome Web Store

Data Collection and Usage

This extension does not collect, store, or transmit any personal or sensitive user data outside your browser.

All favorite chat data is stored locally using Chrome’s storage API and remains on your device. The extension does not send any information to external servers or third parties.

Permissions

  • activeTab: Used only to interact with the currently open ChatGPT tab when the user chooses to add or remove a favorite chat.
  • host permissions: Restricted to chat.openai.com and chatgpt.com to enable adding and displaying favorite chats.
  • scripting: Used to inject a local script that updates UI elements for favorite chats.
  • storage: Used to save favorites locally within Chrome’s storage system.

Third-Party Access

No third-party services, analytics tools, or tracking libraries are used.

Contact

If you have questions or concerns about this privacy policy, you can contact the developer via the Chrome Web Store listing.
