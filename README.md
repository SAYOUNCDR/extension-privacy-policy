# ChatGPT Favorites

A lightweight Chrome extension that lets you **favorite or bookmark ChatGPT conversations** directly in the sidebar.  
Your favorite chats stay saved locally - no accounts, no external servers, no data collection.

[Download from Chrome Web Store](https://chromewebstore.google.com/detail/bfahjhadjkneahhalojpofmbegkllhnj?utm_source=item-share-cb)

## üöÄ Features

- One-click to favorite/unfavorite any ChatGPT chat
- Favorites list appears in the sidebar
- Data stored locally using Chrome storage
- Works on `chat.openai.com` and `chatgpt.com`

## üîí Privacy

This extension **does not collect, share, or transmit** any personal data.  
All favorites are stored locally on your device.

## üß† Permissions Explained

| Permission         | Purpose                                                      |
| ------------------ | ------------------------------------------------------------ |
| `activeTab`        | Accesses the active ChatGPT tab when you click the extension |
| `host_permissions` | Restricts functionality to ChatGPT domains                   |
| `scripting`        | Injects local scripts to modify sidebar UI                   |
| `storage`          | Saves your favorite chat IDs locally                         |
